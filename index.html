Summary: Global Weather Plots



Plots



The purpose of this project was to analyze weather from OpenWeatherMap through their API service and see if we could find any correlations between Latitude and their ensuing weather conditions from about 550 cities.


After assembling the dataset, Matplotlib was used to plot various aspects of weather vs. latitude. Factors analyzed included: temperature, cloudiness, wind-speed and humidity.
